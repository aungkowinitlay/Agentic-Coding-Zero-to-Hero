## 2. Beginner Explanation

Agentic Coding လုပ်ဖို့ AI တစ်ခုတည်းမလုံလောက်ပါဘူး။ AI Agent က project ထဲမှာအလုပ်လုပ်နိုင်ဖို့ အောက်ပါ tools တွေလိုပါတယ်။

```text
Browser        → Website / Web App စမ်းရန်
DevTools       → Console error / Network request စစ်ရန်
VS Code        → Code files ဖတ်/ပြင်ရန်
Terminal       → Commands runရန်
Git            → Project save points / version control
GitHub         → Online backup / collaboration
Node.js + npm  → JavaScript projects runရန်
Vite           → React frontend project scaffold/runရန်
Claude Code    → Agentic coding CLI tool
Codex CLI      → OpenAI coding agent CLI option
```

Mental model—

```text
VS Code = စာရေးခုံ
Terminal = command ပေးတဲ့နေရာ
Git = save game system
Node/npm = JavaScript project engine + package manager
Browser = result စမ်းတဲ့နေရာ
Claude Code / Codex = AI coding agent
```

ဒီ tools တွေကို install လုပ်တာက “tool ထည့်တာ” တစ်ခုတည်းမဟုတ်ပါဘူး။ **အန္တရာယ်မရှိအောင် verify လုပ်တတ်ခြင်း** ပါလိုအပ်တယ်။

# 3. Setup Philosophy

Development tools setup မှာ စည်းကမ်း ၅ ခုထားမယ်။

```text
Rule 1 — Official source မှ install လုပ်ပါ
Rule 2 — Install ပြီး version စစ်ပါ
Rule 3 — Project folder ထဲမှာပဲ agent ကို run ပါ
Rule 4 — Secret/API key တွေ chat/terminal log ထဲမပေါက်အောင်ထိန်းပါ
Rule 5 — Agent ကို command run မခိုင်းခင် command အဓိပ္ပါယ်ကိုရှင်းခိုင်းပါ
```

Beginner အများစုရဲ့အမှားက tool install လုပ်ပြီးတာနဲ့ “အကုန်အဆင်ပြေပြီ” လို့ထင်တာပါ။ တကယ်တော့ development environment မှာ အရေးကြီးတာက—

```text
install
→ verify
→ test command
→ safe workspace
→ Git checkpoint
→ agent permission rules
```

ဆိုတဲ့ workflow ပါ။

# 14. Claude Code Setup

Claude Code ကို ဒီ course ရဲ့ main agentic coding tool အဖြစ်သုံးမယ်။ Claude Code docs မှာ system requirements အနေနဲ့ macOS 13+, Windows 10 1809+ / Windows Server 2019+, Ubuntu 20.04+, Debian 10+, Alpine Linux 3.19+, 4GB+ RAM, x64/ARM64 processor, internet connection, Bash/Zsh/PowerShell/CMD shell လိုအပ်ကြောင်းဖော်ပြထားပါတယ်။ ([Claude][7])

## Install — Native Install Recommended

macOS / Linux / WSL—

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

Windows PowerShell—

```powershell
irm https://claude.ai/install.ps1 | iex
```

Windows CMD—

