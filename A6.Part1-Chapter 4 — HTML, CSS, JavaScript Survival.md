## 2. Beginner Explanation

Web page တစ်ခုကို အိမ်တစ်လုံးနဲ့နှိုင်းယှဉ်ကြည့်ပါ။

```text
HTML       = အိမ်ရဲ့အုတ်မြစ်၊ အခန်းဖွဲ့စည်းပုံ
CSS        = အိမ်ရဲ့အရောင်၊ အလှဆင်၊ layout
JavaScript = မီးခလုတ်၊ တံခါးဖွင့်ပိတ်၊ အလုပ်လုပ်တဲ့ behavior
```

Web page တစ်ခုမှာ—

```text
HTML က “ဘာတွေရှိလဲ” ကိုသတ်မှတ်တယ်။
CSS က “ဘယ်လိုမြင်ရမလဲ” ကိုသတ်မှတ်တယ်။
JavaScript က “ဘာလုပ်မလဲ” ကိုသတ်မှတ်တယ်။
```

ဥပမာ Event Registration page တစ်ခုမှာ—

```text
HTML:
- ခေါင်းစဉ်
- Event name
- Registration form
- Name input
- Phone input
- Register button

CSS:
- Button အရောင်
- Form layout
- Mobile responsive
- Card shadow
- Font size

JavaScript:
- Button နှိပ်တာကိုဖမ်း
- Form data ဖတ်
- Required fields စစ်
- Error message ပြ
- Ticket code ပြ
```

ဒီ mental model ကိုအမြဲမှတ်ပါ။

```text
HTML = Structure
CSS = Style
JavaScript = Behavior
```

# 3. Web Page တစ်ခု Browser ထဲ ဘယ်လိုအလုပ်လုပ်လဲ

Browser က web page တစ်ခုဖွင့်တဲ့အခါ များသောအားဖြင့် ဒီ files ၃ မျိုးကိုဖတ်တယ်။

```text
index.html
style.css
app.js
```

Flow—

```text
Browser ဖွင့်
   ↓
HTML ဖတ်
   ↓
CSS နဲ့ style လုပ်
   ↓
JavaScript နဲ့ behavior ထည့်
   ↓
User မြင်ရတဲ့ page ဖြစ်လာ
```

ဥပမာ—

```text
index.html  → page structure
style.css   → page design
app.js      → button click behavior
```

နောက်ပိုင်း React/Vite သုံးတဲ့အခါ file structure ကပြောင်းမယ်။ ဒါပေမယ့် အခြေခံသဘောတရားက မပြောင်းဘူး။

```text
React component တွေက HTML-like structure ရေးတယ်။
Tailwind/CSS က style ပေးတယ်။
JavaScript က behavior နဲ့ data logic ကိုထိန်းတယ်။
```

# 4. HTML ဆိုတာဘာလဲ

HTML ဆိုတာ **HyperText Markup Language** ပါ။ Beginner အတွက် အလွယ်ဆုံးပြောရရင်—

```text
HTML = Web page ထဲမှာ ဘာတွေရှိမလဲရေးတဲ့ language
```

ဥပမာ—

```html
<h1>Event Management Platform</h1>
<p>လာမည့် event များကိုကြည့်ပြီး register လုပ်နိုင်ပါသည်။</p>
<button>Register</button>
```

ဒီ code ထဲမှာ—

```text
<h1>      → heading
<p>       → paragraph
<button>  → button
```

HTML က web page structure ကိုဖန်တီးတယ်။

---

## 4.1 HTML Element ဆိုတာဘာလဲ

HTML element တစ်ခုက များသောအားဖြင့် opening tag, content, closing tag ပါတယ်။

```html
<p>Hello World</p>
```

ခွဲကြည့်ရင်—

```text
<p>          = opening tag
Hello World  = content
</p>         = closing tag
```

အချို့ tag တွေမှာ closing tag မလိုဘူး။

```html
<img src="event.jpg" alt="Event photo">
<input type="text" name="name">
```

---

## 4.2 HTML Tag အခြေခံများ

အများဆုံးတွေ့မယ့် tags—

| Tag             | အဓိပ္ပါယ်                      |
| --------------- | ------------------------------ |
| `<html>`        | HTML document တစ်ခုလုံး        |
| `<head>`        | page metadata, title, CSS link |
| `<body>`        | user မြင်ရမယ့် content         |
| `<h1>`–`<h6>`   | headings                       |
| `<p>`           | paragraph                      |
| `<a>`           | link                           |
| `<img>`         | image                          |
| `<div>`         | generic container              |
| `<section>`     | section                        |
| `<header>`      | page/header area               |
| `<main>`        | main content                   |
| `<footer>`      | footer                         |
| `<button>`      | button                         |
| `<form>`        | form                           |
| `<input>`       | input field                    |
| `<label>`       | input label                    |
| `<ul>` / `<li>` | list                           |

Beginner အတွက် အရေးကြီးတာက tag အကုန်မှတ်ဖို့မဟုတ်ပါဘူး။ Code ဖတ်တဲ့အခါ “ဒီ tag က page ထဲမှာ ဘာအတွက်လဲ” ခန့်မှန်းနိုင်ရင်လုံလောက်ပါတယ်။

---

## 4.3 HTML Attribute ဆိုတာဘာလဲ

Attribute ဆိုတာ tag ထဲမှာပါတဲ့ extra information ပါ။

ဥပမာ—

```html
<input type="text" id="name" placeholder="သင့်နာမည်ရေးပါ">
```

ဒီမှာ—

```text
type="text"                  → input type
id="name"                    → element id
placeholder="သင့်နာမည်ရေးပါ" → input ထဲမှာပြမယ့် hint
```

နောက်ဥပမာ—

```html
<a href="/events">Event List သို့သွားမည်</a>
```

`href` က link သွားမယ့် path ကိုပြတယ်။

```html
<img src="event.jpg" alt="AI Workshop event photo">
```

`src` က image file path ဖြစ်ပြီး `alt` က image မပေါ်ရင် သို့မဟုတ် screen reader အတွက်ပြမယ့်စာပါ။

---

## 4.4 HTML Document အခြေခံပုံစံ

HTML file တစ်ခုရဲ့ basic structure—

```html
<!DOCTYPE html>
<html lang="my">
  <head>
    <meta charset="UTF-8">
    <title>Event Management Platform</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <main>
      <h1>Event Management Platform</h1>
      <p>လာမည့် event များကိုကြည့်ပြီး register လုပ်နိုင်ပါသည်။</p>
    </main>

    <script src="app.js"></script>
  </body>
</html>
```

အဓိပ္ပါယ်—

```text
<!DOCTYPE html>        → ဒီ file က HTML document ဖြစ်ကြောင်းပြော
<html lang="my">       → မြန်မာဘာသာ content ပါနိုင်ကြောင်းပြ
<head>                 → browser/user မမြင်ရတဲ့ metadata
<meta charset="UTF-8"> → မြန်မာစာ/Unicode support အတွက်အရေးကြီး
<title>                → browser tab title
<link>                 → CSS file ချိတ်
<body>                 → user မြင်ရမယ့် content
<script>               → JavaScript file ချိတ်
```

`meta charset="UTF-8"` ကိုမြန်မာစာ project တွေမှာအရေးကြီးပါတယ်။ မပါရင် တချို့ environment တွေမှာ စာလုံးပြဿနာဖြစ်နိုင်တယ်။

---

## 4.5 Semantic HTML ဆိုတာဘာလဲ

Semantic HTML ဆိုတာ tag ကို meaning ရှိအောင်သုံးတာပါ။

မကောင်းတဲ့နည်း—

```html
<div>Event Management Platform</div>
<div>Register</div>
```

ကောင်းတဲ့နည်း—

```html
<header>
  <h1>Event Management Platform</h1>
</header>

<main>
  <button>Register</button>
</main>
```

Semantic tag တွေသုံးတာက—

```text
Browser နားလည်ရလွယ်တယ်
Screen reader နားလည်ရလွယ်တယ်
Accessibility ကောင်းတယ်
Agent/Developer ဖတ်ရလွယ်တယ်
```

အဓိကမှတ်ရန်—

```text
ခေါင်းစဉ်ဆို h1/h2 သုံး
ခလုတ်ဆို button သုံး
form input အတွက် label သုံး
main content အတွက် main သုံး
```

# 5. HTML Form အခြေခံ

Event Registration project မှာ form အများကြီးသုံးမယ်။

Registration form example—

```html
<form id="registration-form">
  <label for="name">နာမည်</label>
  <input id="name" name="name" type="text" required>

  <label for="phone">ဖုန်းနံပါတ်</label>
  <input id="phone" name="phone" type="tel" required>

  <label for="email">အီးမေးလ်</label>
  <input id="email" name="email" type="email">

  <button type="submit">Register လုပ်မည်</button>
</form>
```

အရေးကြီးတဲ့အချက်—

```text
<label for="name"> နဲ့ <input id="name"> ကချိတ်ထားတယ်။
required ဆိုရင် field မဖြည့်ဘဲ submit လုပ်လို့မရဘူး။
button type="submit" ဆိုရင် form submit လုပ်မယ်။
```

Form တွေမှာ accessibility အတွက် label ပါရမယ်။ Placeholder ကို label အစားမသုံးသင့်ပါ။

မကောင်းတဲ့ form—

```html
<input placeholder="Name">
```

ပိုကောင်းတဲ့ form—

```html
<label for="name">နာမည်</label>
<input id="name" name="name" placeholder="ဥပမာ - မောင်မောင်">
```

# 6. CSS ဆိုတာဘာလဲ

CSS ဆိုတာ **Cascading Style Sheets** ပါ။ Beginner အတွက်—

```text
CSS = Web page ကိုဘယ်လိုမြင်ရမလဲ သတ်မှတ်တဲ့ language
```

ဥပမာ—

```css
h1 {
  color: blue;
  font-size: 32px;
}

button {
  padding: 12px 16px;
  border-radius: 8px;
}
```

ဒီ code က—

```text
h1 ကို အပြာရောင်နဲ့ 32px size ပြ
button ကို padding နဲ့ round corner ပေး
```

လို့ပြောတာပါ။

---

## 6.1 CSS Rule Anatomy

CSS rule တစ်ခုက selector, property, value ပါတယ်။

```css
.event-card {
  background: white;
  padding: 16px;
  border-radius: 12px;
}
```

ခွဲကြည့်ရင်—

```text
.event-card  = selector
background   = property
white        = value
padding      = property
16px         = value
```

အဓိပ္ပါယ်—

```text
event-card class ပါတဲ့ element တွေကို style ပေးပါ
```

---

## 6.2 CSS Selector အခြေခံ

### Tag selector

```css
p {
  color: #333;
}
```

`<p>` tags အားလုံးကို style ပေးမယ်။

### Class selector

```css
.event-card {
  border: 1px solid #ddd;
}
```

`class="event-card"` ပါတဲ့ elements တွေကို style ပေးမယ်။

HTML—

```html
<div class="event-card">
  <h2>AI Workshop</h2>
</div>
```

### ID selector

```css
#registration-form {
  max-width: 480px;
}
```

`id="registration-form"` ပါတဲ့ element ကို style ပေးမယ်။

Beginner rule—

```text
Reusable style အတွက် class ကိုသုံးပါ။
Unique element အတွက် id ကိုသုံးပါ။
```

React/Tailwind သုံးလာတဲ့အခါ class concept အလွန်အရေးကြီးလာမယ်။

---

## 6.3 CSS Class ဆိုတာဘာလဲ

Class က HTML element ကို style group တစ်ခုနဲ့ချိတ်ပေးတာပါ။

HTML—

```html
<button class="primary-button">Register လုပ်မည်</button>
```

CSS—

```css
.primary-button {
  background: #2563eb;
  color: white;
  padding: 12px 16px;
  border-radius: 8px;
}
```

Class နာမည်က meaning ရှိသင့်တယ်။

ကောင်းတဲ့ class names—

```text
event-card
primary-button
registration-form
error-message
page-header
```

မကောင်းတဲ့ class names—

```text
blue-big-thing
abc
new-style
box1
```

---

## 6.4 Box Model

Browser ထဲက element တစ်ခုချင်းစီကို box တစ်ခုလိုစဉ်းစားနိုင်တယ်။

```text
Content
Padding
Border
Margin
```

အလွယ်ဆုံးမှတ်ရန်—

```text
content = အတွင်းကစာ/ပုံ
padding = content နဲ့ border ကြားအတွင်းအကွာအဝေး
border  = အနားသတ်
margin  = တခြား element နဲ့အပြင်အကွာအဝေး
```

Example—

```css
.event-card {
  padding: 16px;
  border: 1px solid #ddd;
  margin-bottom: 16px;
}
```

အဓိပ္ပါယ်—

```text
Card အတွင်းအကွာ 16px
အနားသတ် 1px
အောက်ဘက် card နဲ့အကွာ 16px
```

## 6.6 Responsive Design ဆိုတာဘာလဲ

Responsive design ဆိုတာ screen size အလိုက် UI ပုံစံပြောင်းနိုင်တာပါ။

ဥပမာ—

```text
Phone မှာ event card ၁ ကော်လံ
Tablet မှာ ၂ ကော်လံ
Desktop မှာ ၃ ကော်လံ
```

CSS media query example—

```css
.event-list {
  display: grid;
  gap: 16px;
  grid-template-columns: 1fr;
}

@media (min-width: 768px) {
  .event-list {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (min-width: 1024px) {
  .event-list {
    grid-template-columns: repeat(3, 1fr);
  }
}
```

အဓိပ္ပါယ်—

```text
Default = 1 column
768px ထက်ကျယ်ရင် = 2 columns
1024px ထက်ကျယ်ရင် = 3 columns
```

Beginner အတွက်အရေးကြီးဆုံး—

```text
Mobile-first စဉ်းစားပါ။
အရင် phone အတွက်ရေးပြီး desktop ကိုတိုးချဲ့ပါ။
```

# 7. JavaScript ဆိုတာဘာလဲ

JavaScript ဆိုတာ web page ကို interactive ဖြစ်အောင်လုပ်တဲ့ programming language ပါ။

Beginner အတွက်—

```text
JavaScript = User action ကိုတုံ့ပြန်ပြီး page behavior ကိုထိန်းတဲ့ language
```

ဥပမာ—

## Step 4 — app.js ထဲရေးပါ

```js
const form = document.querySelector("#registration-form");
const nameInput = document.querySelector("#name");
const phoneInput = document.querySelector("#phone");
const message = document.querySelector("#message");

form.addEventListener("submit", function (event) {
  event.preventDefault();

  const name = nameInput.value.trim();
  const phone = phoneInput.value.trim();

  message.className = "message";

  if (name === "") {
    message.textContent = "နာမည်ဖြည့်ပါ။";
    message.classList.add("error");
    return;
  }

  if (phone === "") {
    message.textContent = "ဖုန်းနံပါတ်ဖြည့်ပါ။";
    message.classList.add("error");
    return;
  }

  message.textContent = `${name} အတွက် registration စတင်ပြီးပါပြီ။ Ticket code ကို နောက်အဆင့်မှာထုတ်ပေးပါမည်။`;
  message.classList.add("success");
});
```

ဒီ file ထဲမှာ JavaScript behavior ကိုရေးထားတယ်။

ပါဝင်တာ—

```text
Form submit event ဖမ်း
Name/phone value ဖတ်
Empty ဖြစ်ရင် error ပြ
အောင်မြင်ရင် success message ပြ
```

# 17. Common Errors

## Error 1 — HTML tag မပိတ်ခြင်း

မကောင်းတဲ့ code—

```html
<p>Hello
<div>World</div>
```

ပြဿနာ—

```text
Browser က guess လုပ်ပြီး render လုပ်နိုင်ပေမယ့် layout ပျက်နိုင်တယ်။
```

ကောင်းတဲ့ code—

```html
<p>Hello</p>
<div>World</div>
```

---

## Error 2 — CSS file မချိတ်မိခြင်း

HTML ထဲမှာ—

```html
<link rel="stylesheet" href="style.css">
```

မပါရင် CSS မအလုပ်လုပ်ဘူး။

စစ်ရန်—

```text
style.css file name မှန်လား
index.html နဲ့ same folder ထဲရှိလား
href path မှန်လား
```

---

## Error 3 — JavaScript file မချိတ်မိခြင်း

HTML ထဲမှာ—

```html
<script src="app.js"></script>
```

မပါရင် JavaScript မအလုပ်လုပ်ဘူး။

စစ်ရန်—

```text
app.js file name မှန်လား
script tag ပါလား
path မှန်လား
browser console error ရှိလား
```

---

## Error 4 — querySelector မှာ id/class မှားခြင်း

HTML—

```html
<button id="register-button">Register</button>
```

JavaScript မှာမှားရေး—

```js
const button = document.querySelector("#register");
```

ပြဿနာ—

```text
#register ဆိုတဲ့ id မရှိလို့ button ကိုမတွေ့ဘူး။
```

မှန်တဲ့ code—

```js
const button = document.querySelector("#register-button");
```

---

## Error 5 — `#` နဲ့ `.` ရောထွေးခြင်း

CSS/JavaScript selector မှာ—

```text
#name       → id="name"
.name       → class="name"
button      → <button> tag
```

HTML—

```html
<p id="message" class="error-message"></p>
```

Selector—

```css
#message {
  color: green;
}

.error-message {
  font-weight: bold;
}
```

---

## Error 6 — Form submit ဖြစ်ပြီး page refresh ဖြစ်ခြင်း

JavaScript ထဲမှာ—

```js
event.preventDefault();
```

မပါရင် form submit လုပ်တဲ့အခါ page refresh ဖြစ်နိုင်တယ်။

Form JS example—

```js
form.addEventListener("submit", function (event) {
  event.preventDefault();
});
```

---

## Error 7 — Browser console error မကြည့်ခြင်း

JavaScript error တက်ရင် page ကဘာမှမပြနိုင်ဘူး။ Browser DevTools Console ကိုကြည့်ပါ။

Common error—

```text
Cannot read properties of null
```

အဓိပ္ပါယ်—

```text
JavaScript က element တစ်ခုရှာတာ မတွေ့ဘူး။
Selector/id/class/path မှားနိုင်တယ်။
```

---

## Error 8 — Placeholder ကို label အစားသုံးခြင်း

မကောင်းတဲ့ form—

```html
<input placeholder="နာမည်">
```

ပိုကောင်းတဲ့ form—

```html
<label for="name">နာမည်</label>
<input id="name" placeholder="ဥပမာ - မောင်မောင်">
```

Accessibility အတွက် label ပါရမယ်။
